## API Biblioteca Laravel
<p>API para gestionar los libros en bibliotecas, prestarlos, ver el stock y regresarlos.</p>

`$ vagrant up`<br>
`[GET] http://biblioteca.laravel/api/bibliotecas`<br>
`[POST] http://biblioteca.laravel/api/bibliotecas`<br>
`[GET] http://biblioteca.laravel/api/librios`<br>
`[POST] http://biblioteca.laravel/api/librios`<br>
`[GET] http://biblioteca.laravel/api/usuarios`<br>
`[POST] http://biblioteca.laravel/api/usuarios`<br>
`[GET] http://biblioteca.laravel/api/pedidos`<br>
`[POST] http://biblioteca.laravel/api/pedidos/usuario/{usuario_id}/libro/{libro_id}`<br>
`[PUT] http://biblioteca.laravel/api/entregas/usuario/{usuario_id}/libro/{libro_id}`
